> **Status: planned — Milestone M8.**
> **Distribution: `bundled-private`.** Not on npm. Ships inside the desktop
> bundle. See [distribution tiers](../architecture/distribution-tiers).

Sync workspace config, lockfile, and selected runtime artifacts across
devices and team members.

## Targeted scope

- Push / pull workspaces from a hosted control plane.
- Encrypted backup + restore.
- Multi-device editing (paired with [`os-collab`](./os-collab) for live
  co-editing).
- Self-host the control plane day one.
