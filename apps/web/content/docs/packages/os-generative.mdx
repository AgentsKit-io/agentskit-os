> **Status: planned — Milestone M7.**
> **Distribution: `bundled-private`.** Not on npm. Ships inside the desktop
> bundle. See [distribution tiers](../architecture/distribution-tiers).

Natural-language synthesis of AgentsKitOS artifacts.

- NL → agent definition
- NL → flow DAG (with run-mode + cost preview)
- NL → trigger spec
- NL → tool wrapper

Output is always **editable** code or YAML — never opaque generated blobs.
Every generation produces a Zod-validated artifact you can read, diff, and
ship through the normal review process.
