> **Status: planned — Milestone M2.**
> **Distribution: `public`** (when shipped). Embedders can install it.

Reusable UI primitives shared across `os-desktop` and any custom OS surface.

## Components (target)

- **Dashboard** — workspace overview, cost heat map, run feed.
- **FlowEditor** — DAG canvas with node inspector, run-mode preview, branch-from-step.
- **TraceViewer** — span tree with replay, diff, anomaly highlights.
- **CommandPalette** — Cmd-K with workspace-scoped actions.

Built with shadcn + Tailwind so embedders can theme freely.

## Install (when published)

```bash
pnpm add @agentskit/os-ui
```
