## Shipping (M1)

| Package | Purpose |
|---|---|
| [`@agentskit/os-core`](./os-core) | Zod contracts, event bus, errors, workspace model |
| [`@agentskit/os-cli`](./os-cli) | init, doctor, run, lock, publish, deploy |
| [`@agentskit/os-flow`](./os-flow) | DAG executor, durable, cycle-safe, run-mode-aware |
| [`@agentskit/os-runtime`](./os-runtime) | Handler factories with pluggable LLM/tool/memory adapters |
| [`@agentskit/os-runtime-agentskit`](./os-runtime-agentskit) | AgentsKit binding for os-runtime |
| [`@agentskit/os-storage`](./os-storage) | File-backed CheckpointStore + lockfile reader/writer |
| [`@agentskit/os-audit`](./os-audit) | Merkle batch chain emitter |
| [`@agentskit/os-plugins`](./os-plugins) | Plugin loader, manifest validation, capability checks |
| [`@agentskit/os-sandbox`](./os-sandbox) | Built-in `none` and `process` runtimes; container/vm via plugins |
| [`@agentskit/os-templates`](./os-templates) | Starter template gallery |
| [`@agentskit/os-import`](./os-import) | Langflow / n8n / Dify → ConfigRoot importers |
| [`@agentskit/os-marketplace-sdk`](./os-marketplace-sdk) | Manifest builder, integrity hashing, signing, bundle metadata |
