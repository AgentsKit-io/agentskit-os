## Why we're slower on purpose

Existing agent OSes — Flowise, Langflow, n8n, Dify — optimized
**speed-of-shipping** over contract stability. The result: drift, lock-in,
abandoned plugins, and ecosystems that fragment after every major release.

AgentsKitOS optimizes the opposite axis. Slower visible progress; longer-lived
ecosystem. Stable contracts, public ADRs, RFCs before breaking changes,
SemVer-strict releases.
