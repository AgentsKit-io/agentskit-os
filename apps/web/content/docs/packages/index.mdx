Each package declares a **distribution tier** ([ADR-0014](../architecture/distribution-tiers)):

- **`public`** — published to npm. `pnpm add @agentskit/os-…` works.
- **`bundled-private`** — ships inside the desktop bundle only. Not on npm.
- **`internal-only`** — tooling / fixtures. Never shipped.

End users never run `npm install`. They get a signed Tauri bundle.
Plugin authors and embedders install only the **public** packages.
