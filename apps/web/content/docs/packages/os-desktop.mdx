> **Status: planned — Milestone M2.**

A native cross-platform shell built on Tauri 2.

## Targeted scope

- &lt;15 MB installer.
- &lt;800 ms cold start.
- Bundles `os-ui` (FlowEditor, TraceViewer, Dashboard, CommandPalette).
- Self-host day one. Air-gap supported.
- Windows · macOS · Linux.

The desktop is one OS surface among many. Everything you can do in the
desktop, you can do via the CLI or programmatically through `os-core` +
`os-runtime`.
